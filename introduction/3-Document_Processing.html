Document Processing

A user can submit documents of different content types for processing. Below you will find three different methods to submit a document for processing:

Process a document by content type:

Method 1: Multipart/form-data upload

Submit a request to process a multipart/form-data file upload.

Method 2: Base64 encoded file upload

Submit a request to process a Base64 encoded document.

Method 3: Specify a file URL

Submit a request to process a file using a URL.

Retrieving extracted data by response types:

Synchronous response

All processing requests are synchronous by default. Therefore, a user that submits a POST request for document processing will receive the extracted data in the API response.

Asynchronous response

Asynchronous processing requests receive an immediate response. However, the data extraction runs in a background process. Once data extraction completes, Veryfi makes a request to your configured webhook URL. The webhook URL is configurable in the Keys section of Settings.

    • Uploading Zip files

    Uploading a zip file as a multipart/form-data is the fastest way to upload files.

    Tip 
     Uploading a zip file as a multipart/form-data is the fastest way to upload files
    How to upload a zip file

    1. Create a zip file containing your files for processing

    2. Configure Veryfi Client with your CLIENT_ID, CLIENT_SECRET, USERNAME and API_KEY

    3. Submit the zip file with Veryfi Client for processing

    Uploading a Zip File
    • PYTHON
    1import zipfile
2from veryfi import Client
3
4list_files = ['receipt1.jpg', 'receipt2.jpg']
5zip_file_path = 'receipts.zip'
6
7with zipfile.ZipFile(zip_file_path, 'w') as zipF:
8    for file in list_files:
9        zipF.write(file, compress_type=zipfile.ZIP_DEFLATED)
10
11client_id = 'your_client_id'
12client_secret = 'your_client_secret'
13username = 'your_username'
14api_key = 'your_password'
15
16veryfi_client = Client(client_id, client_secret, username, api_key)
17response = veryfi_client.process_document(zip_file_path)
    How to manually upload a Base64 encoded file

    1. Compress the files

    2. Create a Base64 encoded string from the data

    3. Configure Client ID and Authorization headers

    4. Configure Document Processing request parameters.

    5. Submit the POST request for processing

    Processing in-memory compressed files
    • PYTHON
    1import io
2import zipfile
3import base64
4import requests
5import json
6
7list_files = ['receipt1.jpg', 'receipt2.jpg']
8
9zip_buffer = io.BytesIO()
10with zipfile.ZipFile(zip_buffer, "a", zipfile.ZIP_DEFLATED, False) as zip_file:
11    for file_name in list_files:
12        with open(file_name, "rb") as image_file:
13            zip_file.writestr(file_name, image_file.read())
14
15encode_zip_string = base64.b64encode(zip_buffer.getvalue())
16
17client_id = 'your_client_id'
18client_secret = 'your_client_secret'
19username = 'your_username'
20api_key = 'your_password'
21
22headers = {
23    "User-Agent": "Python Veryfi-Python/3.0.0",
24    "Accept": "application/json",
25    "Content-Type": "application/json",
26    "Client-Id": client_id,
27    "Authorization": f"apikey {username}:{api_key}"
28}
29api_url = "api.veryfi.com/api/v8/partner/documents"
30request_arguments = {
31    "file_name": file_name,
32    "file_data": encode_zip_string,
33}
34_session = requests.Session()
35response = _session.request(
36            "POST",
37            url=api_url,
38            headers=headers,
39            data=json.dumps(request_arguments),
40        )
